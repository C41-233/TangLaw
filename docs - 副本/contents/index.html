序言

天、地、人三才确立，宇宙间的万事万物也就随之确立。在一切有生命、有智慧的生物钟，人是最尊贵的。没有一个国家不是依靠百姓而设立君主和官吏，通过政令教化来施行刑法的。德礼为政教之本，刑罚为政教之用。如果有人放纵私欲、愚昧无知，或者心智蒙昧而犯下罪过，大则扰乱天下，小则破坏各种规章制度。因此，国家必须建立完善的法律制度，以刑止刑，以杀止杀。大唐皇帝凭先皇巩固的基业，英明的声威践行先圣踪迹，施恩泽于万物，宽刑罚于百姓。为明确犯罪行为，统一刑罚尺度，由皇帝敕令，经宰辅台省编撰制定本法，施行于全国。

第一条 笞刑

笞刑分为五等：

（一）笞一十，赎铜一斤；

（二）笞二十，赎铜二斤；

（三）笞三十，赎铜三斤；

（四）笞四十，赎铜四斤；

（五）笞五十，赎铜五斤。
