# 项目

本项目是《唐律疏议》的注解网站，通过generator工具将src的内容转化输出到docs，网站入口在docs/index.html。

## 目录结构

- references 参考文献，不应当读取
- src 原始文档
- docs 输出的静态网站页面
- generator 一个dotnet项目，用于将src的原始文本内容转化输出到docs

### src的目录结构

src包含序言、正文、附录、释义。

序言只有 `0.xml`，是序言的全部内容。

### docs的目录结构

index.html是网站入口，其结构包括：

- 标题 《唐律疏议》的理解与适用
- 正文目录 包含序言和正文的目录，正文包括篇、章、条，点击可以进入条文页面。
- 律文，包含序言和正文的律条。
- 附录目录。
- 释义目录。

## 基础规则

- 不允许执行任何 `git `命令。
- 不允许删除任何文件。
