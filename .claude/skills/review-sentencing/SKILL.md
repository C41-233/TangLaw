# 罪名量刑一致性审查

本 skill 用于审查《唐律疏议》注解项目中五个来源之间的罪行、法定刑（量刑）、十恶标签、法条引用、规则描述是否一致。

## 审查范围

| 来源 | 路径 | 说明 |
|------|------|------|
| 亲属释义 | `src/释义/亲属/` | 每个亲属关系的个人犯罪表 |
| 尊卑释义 | `src/释义/尊卑/` | 部曲、奴婢对主人及亲族的犯罪表 |
| 罪刑表 | `src/附录/3-罪刑表/` | 按罪名分类的罪刑对应表 |
| 量刑表 | `src/附录/4-量刑表/` | 按刑罚等级分类的罪刑对应表 |
| 缘坐 | `src/释义/Y/缘坐.xml` | 缘坐制度汇总表 |

## 审查维度

对每一组对应条目，逐一比对以下五个字段：

1. **罪行（罪名/行为描述）** — 同一犯罪行为的命名和表述是否统一
2. **法定刑（量刑）** — 刑罚是否一致（斩、绞、流XX里、徒X年、杖XX、笞XX）
3. **十恶标签** — 是否属于十恶（谋反、谋大逆、谋叛、恶逆、不道、大不敬、不孝、不睦、不义、内乱），标签是否一致
4. **法条引用** — 引用的律典条文编号是否一致
5. **规则描述** — 对复合规则（如"减一等"、"加一等"、"以诬告凡人论"等）的措辞是否一致

## 审查流程

### 第一步：读取所有源文件

1. 读取 `src/释义/亲属/` 下所有 `.xml` 文件，提取各亲属关系表中的犯罪条目
2. 读取 `src/释义/尊卑/` 下所有 `.xml` 文件，提取部曲/奴婢对主人的犯罪条目
3. 读取 `src/附录/3-罪刑表/` 下所有罪名分类文件
4. 读取 `src/附录/4-量刑表/` 下所有刑罚分类文件
5. 读取 `src/释义/Y/缘坐.xml`

### 第二步：逐个罪行进行交叉比对

按犯罪类型分组，每组内部比较五个来源中对应条目的五个字段是否一致。犯罪类型包括但不限于：

- **职制律**：闻丧匿不举哀、闻丧择日举哀、居丧释服从吉、居丧玩乐（忘哀作舞乐/作杂戏/听奏乐/参加吉席）
- **户婚律**：居丧嫁娶（嫁娶妻/嫁娶妾）
- **贼盗律**：谋杀、厌魅（意图杀人/意图致人疾苦/为求爱媚）、诅咒（意图杀人/意图致人疾苦/为求爱媚）
- **斗讼律**：殴打、过失杀、过失伤、詈骂、告、诬告
- **缘坐**：谋反、大逆、叛国、杀一家非死罪三人、肢解人、造畜蛊毒

### 第三步：逐组比对方法

对于每个犯罪类型：

1. 从**罪刑表**（`src/附录/3-罪刑表/`）中提取该罪行的完整客体→刑罚映射表
2. 与**亲属文件**中对应亲属的表格逐行比对（客体、法定刑、十恶、法条、规则描述）
3. 与**量刑表**（`src/附录/4-量刑表/`）中对应刑罚等级的条目逐行比对
4. 与**尊卑文件**中对应主人的犯罪表逐行比对
5. 与**缘坐.xml**中的缘坐数据比对

#### 比对优先级

| 对比方向 | 基准来源 | 作用 |
|---------|---------|------|
| 亲属文件 ↔ 罪刑表 | 以罪刑表为准，检查亲属文件是否有遗漏或错误 |
| 量刑表 ↔ 罪刑表 | 双向比对，确保覆盖完备且数据一致 |
| 尊卑文件 ↔ 罪刑表+量刑表 | 以罪刑表/量刑表为准 |
| 缘坐.xml ↔ 各亲属文件+量刑表 | 逐行比对，确保范围、法定刑、法条一致 |

### 第四步：特别关注的高风险点

根据历史审查经验，以下位置最容易出现不一致：

1. **缌麻尊长/卑幼表格** — 此亲等范围最宽，条目最多，容易出现格式错误或占位符
2. **卑幼诬告法条编号** — 告与诬告的法条编号不同（告为 347，诬告为 347.1），容易混淆
3. **主人系列谋杀法条** — 凡人谋杀（256）、谋杀期亲尊长（253）、部曲奴婢谋杀主（254）三组易混
4. **诅咒在量刑表中的覆盖** — 诅咒与厌魅刑罚相同，但易被遗漏
5. **厌魅与诅咒的十恶标签差异** — 厌魅基本犯属于"不道"，诅咒基本犯不属于十恶（仅致死时入"恶逆"），两罪十恶标签不可混淆
6. **规则描述措辞统一** — "诬告之罪加一等" vs "所告之罪加一等"；"反坐" vs "以诬告凡人论"
7. **居丧玩乐各子类型的区分** — 忘哀作舞乐/作杂戏/听奏乐/参加吉席有不同的刑罚

### 第五步：输出审查报告

报告格式如下：

```
# 罪名量刑一致性审查报告

## 已确认一致（N项）
逐项列出已验证一致的内容。

## 不一致（N项）
每项包含：
- **问题**: 简要描述
- **涉及文件**: 具体文件路径及行号
- **冲突值**: 各来源当前值的对比表
- **建议**: 正确的值是什么（如果可确定）

## 审查范围
本次审查覆盖的目录/文件清单。
```

**审查后不得直接修改文件。** 报告完成后应询问用户是否同意修改，待用户确认后再执行编辑。

## 重要注意事项

- 法条编号 253/254/256 之间的区别要特别注意：
  - 253 = 谋杀期亲尊长、缌麻尊长
  - 254 = 部曲奴婢谋杀主
  - 256 = 凡人谋杀
- 诬告法条编号规则：
  - 告尊长 = 346，诬告尊长 = 346.1
  - 告卑幼 = 347，诬告卑幼 = 347.1
  - 告主人 = 349，诬告主人亲族 = 349.1
- 对于非纯文本的表格单元格（含 `<br/>` 标签的），比较时应忽略标签差异，仅比内容
- 十恶标签中，厌魅基本犯始终带"不道"，诅咒基本犯不带十恶，仅有"恶逆"（对父母/祖父母）或致死时入"恶逆"
